# global clicker
this is a global clicker app made with django for hack club's [onekey!](https://onekey.hackclub.com)

## what even *is* a global clicker?
basically, it's a clicker except anyone from anywhere can increment the clicks. there's also a heatmap of where all the clicks are originating from (yes, this requires your location. i SWEAR i'm not shady.)
